# Library App

A simple and clean Library Management Application built using **Node.js**, **Express**, **TypeScript**, and **MySQL**.  
This project provides a basic system to manage books, authors, and categories with full CRUD functionality.  
The architecture follows a modular and organized structure to keep the code easy to read, maintain, and extend.

You can create books, assign authors and categories, update records, view all items, and delete them.
